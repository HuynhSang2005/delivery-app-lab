# Suggested Commands

## Development
- `bun install` - Install all dependencies
- `bun dev` - Start API dev server
- `bun run build` - Production build
- `bun typecheck` - TypeScript strict check
- `bun test` - Run tests
- `bun lint` - Run linter

## Database
- `bunx --bun prisma generate` - Generate Prisma client
- `bunx --bun prisma migrate dev` - Run migrations
- `bunx --bun prisma studio` - Open Prisma Studio

## Mobile
- `bunx expo start` - Start Expo dev server

## Git (Windows)
- `git status` - Check status
- `git diff` - See changes
- `git log --oneline -10` - Recent commits

## Task Tracking (beads)
- `bd ready --json` - Find unblocked work
- `bd list --status open --json` - Open issues
- `bd create "title" -t task --json` - Create task
